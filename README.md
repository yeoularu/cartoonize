## Get Started

`Python3` and `OpenCV` are needed.

```bash
git clone https://github.com/yeoularu/cartoonize
cd cartoonize
```

Edit `video_source` at `cartoonize.py` (Camera ID or File path). Default is `'sample.mp4'`

```bash
python cartoonize.py
```

- Press the Esc key to exit.
